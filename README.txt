Aurora City FC Shared Data Pack

Upload these five files together in the same GitHub folder:

- AuroraFCData.js
- AuroraCityFC_ManagerDashboard.html
- AuroraCityFC_TransferCentre.html
- AuroraCityFC_Boardroom.html
- AuroraCityFC_SquadHub.html

To update the shared FC system later, edit AuroraFCData.js first:
- targets.incomeFour
- targets.upgradeFour
- budget values
- dataUrl
- page links
- platformRules
